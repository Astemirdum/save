# http client-server web service - file service

## The following concepts are applied in app:
- The Clean Architecture approach. DI
- Graceful Shutdown.
- Running app in docker containers (docker-compose).
- CI (GitHub Action)


### to start the server:

#### to run app in docker containers
```
make run
```

```
make run-server
make run-client
```

#### to run lint
```
make lint
```

#### to run tests
```
make test
```
